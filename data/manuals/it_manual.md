# IT Troubleshooting Manual

## 1. VPN Access Issues
If you cannot connect to the Enterprise VPN:
1. Ensure your internet connection is stable.
2. Restart the Cisco AnyConnect client.
3. Verify your MFA token is synchronized.
4. If the issue persists, open a ticket on the Jira Service Desk under the "Network Access" category.

## 2. Software Installation
Standard software (e.g., MS Office, Slack, Zoom) can be installed via the self-service portal (Company Portal on Windows, Jamf Self Service on Mac). For custom software, manager approval is required before IT can provide local admin rights.

## 3. Hardware Replacement
Laptops are eligible for replacement every 3 years. If a device is damaged, a police report (if stolen) or an incident report must be filed before a replacement is issued.
